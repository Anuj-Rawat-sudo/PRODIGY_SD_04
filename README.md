# Sudoku Solver using Backtracking

## Overview

This project is a GUI-based Sudoku Solver developed using Python and Tkinter. It uses the Backtracking Algorithm to solve any valid 9×9 Sudoku puzzle efficiently.

## Features

- GUI Interface
- Editable Sudoku Grid
- Load Sample Puzzle
- Solve Sudoku
- Clear Board
- Displays Solving Time
- Input Validation
- User Friendly

## Technologies

- Python
- Tkinter

## Algorithm

Backtracking

## Time Complexity

Worst Case:
O(9^(81))

## Space Complexity

O(81)

## How to Run

python sudoku_solver.py
